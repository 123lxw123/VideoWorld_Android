# VideoWorld_Android
本项目实现了资源列表、详情、搜索，切换资源库、搜索引擎，跳转第三方软件下载、热修复、推送分享等核心功能，应用 [Retrofit2][1]+[RxJava2][2]+[Glide][3] 等主流框架。二次封装网络请求框架，实现异常和错误的统一处理，集成微信分享、极光推送、阿里第二代热修复框架 [HotFix2][4]。

# 功能列表  


- [x] 支持在线播放视频
- [x] 支持视频边下边播
- [x] 支持本地播放视频
- [x] 支持视频列表播放
- [x] 支持收藏视频资源
- [x] 支持播放历史记录
- [x] 支持磁力链接搜索
- [x] 支持网络请求缓存
- [x] 支持多种主题随意切换
- [x] 支持多个资源库随意切换
